SSTF Disk Scheduling

Introduction to SSTF Disk Scheduling

Shortest Seek Time First (SSTF) is a disk scheduling algorithm used to optimize the seek time of a disk drive. The seek time is the time taken by the disk arm to move to the desired track to read or write data.

In SSTF, the operating system selects the disk I/O request that requires the least disk arm movement from its current position to the requested position. By selecting the closest request to the current position, SSTF aims to minimize the seek time and improve the overall disk performance.

SSTF is a type of Elevator Algorithm, which is designed to efficiently handle requests in both directions of disk movement. It belongs to a family of disk scheduling algorithms that includes SCAN and C-SCAN.
